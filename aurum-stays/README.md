# AURUM STAYS

Premium luxury rental platform built with Next.js 14, TypeScript, Tailwind CSS & Prisma.

## Quick Start
```bash
npm install
cp .env.example .env   # then edit DATABASE_URL + JWT_SECRET
npx prisma db push
npm run db:seed
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
